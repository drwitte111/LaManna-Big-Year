Admin Panel
Manage family accounts and Big Year settings
Big Year window

Set the start and end dates for the Big Year. Times are entered in EDT (Eastern Daylight Time). Bird log entries are only allowed between these two moments.

Data export
Export all bird entries
Downloads a CSV with every sighting across all family members — user, bird, location, date, note, and points.
Location fix
Re-geocode all bird locations
Loops through every bird entry that has a location but missing or mismatched coordinates and updates lat/lng from the city database. Only updates entries where coordinates are null or the city lookup returns a different result.
Unlisted species

Birds someone logged that aren't on the current Ohio species list (ohio_birds.txt on GitHub). Use "Edit" to fix a typo/rename right here, or treat this as a guide and update the GitHub file directly.

All bird entries

Every sighting logged by anyone. Search, edit, or delete an individual entry.
